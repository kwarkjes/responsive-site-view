About

This Chrome extension helps web designers do two things..

Quickly preview responsive web designs at the dimensions they will be seen at on popular mobile devices.

Secondly, to grab an image for demonstration purposes and for use in portfolios.

As well as enhancing the extension further, we're hoping to improve the device list. If there are any devices you think should be included or removed please contact us via our website or on twitter.

Please be aware that this extension does not emulate the devices listed. It simply renders websites via the Chrome browser at the specified dimensions. These are the dimensions a responsive website might appear at on the selected device, assuming it used something like the following meta element in the header:

<meta name="viewport" content="width=device-width, initial-scale=1.0" />

You will still need to test your website on lots of real devices. Why not start or contribute to a local device library? See LabUp! for more details..

This extension is by NimbleWorks in association with the Exeter Web device library.
